NO JS, Pure CSS3/HTML Carousel.

Slide 1

No JS here.

This guy is running on CSS and HTML

Great for progressive enhancement or simple one-off carousels!

Slide 2

Curious how it works?

It is actually pretty simple.

The magic lies withing radio inputs and css3 animations!

Slide 3

Each slide is assigned it's own radio input.

When an input's status is "checked", it is positioned within the visible window on the inner container.

Slide 4

You can't see the actual radio buttons though! They are all set to "display: none;"

So now you're probably thinking "How can you click something that is hidden?"

Labels!

Slide 5

The previous & next arrows + the pagination buttons are all labels associated with a slide.

For each slide there are 2 labels

1 label to control the previous and next functionality, and another label to control pagination.

Slide 6

That sounds confusing right? Don't worry, it is pretty simple.

Right now the 6th radio button is checked (we're on slide 6). So use the "nth-child(5)" pseudo-selector to style the previous button, and the "nth-child(7)" pseudo-selector to style the next button.

Each time a slide is checked and showed, the previous and next slides' labels are show accordingly.

Slide 7

Pagination works the same way.

When a radio button is checked, it's pagination label is styled to active.

Conversely, the other slides are unchecked and show the default style.

Throw some animation on the inner container and boom, you're done!

NO JS, Pure CSS3/HTML LightBox.

This version of the pure CSS/HTML lightbox also uses radio buttons to toggle activation. Each image thumbnail is wrapped in a label. When checked, the large version fades in. The trick here is to set a default "checked" on the background overlay to hide it. Meaning, when no image is checked, the overlay is hidden.
